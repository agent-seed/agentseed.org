Minecraft Server Status

Minecraft Server Status is a Discord bot that can ping both Java and Bedrock Edition Minecraft servers. Get information about the status of your favorite servers with a simple command.

Minecraft Server Status is a Discord bot that can ping both Java and Bedrock Edition Minecraft servers. Get information about the status of your favorite servers with a simple command.


Easy to use commands

Change the bot's prefix in your guild

m/ save prefix [your prefix]

Send an automatically refreshing message

m/ automsg

Check the status of a saved server

m/ check

Check Java Edition server:

m/ status [Server IP] [Optional Port]

Correct usage:

m/ status mc.hypixel.net 25565

m/ status mc.hypixel.net:25565

Bedrock Edition server:

m/ bedrock [Server IP] [Optional Port]

Correct usage:

m/ bedrock 142.44.169.167 2087

m/ bedrock 142.44.169.167:2087

Add a server to the database (one per guild)

m/ save server [type] [server ip] [optional port]

Correct usage:

m/ save server bedrock 142.44.169.167:2087

m/ save server java mc.hypixel.net
Remember to put a space in between m/ and the command. Either a space or a colon can be used to specify the port number. The port number is only required when the default (25565 for Java and 19132 for Bedrock) is not used.

Frequently Asked Questions

Can the bot check the status of realms?

No, the bot can only be used to check traditional hosted Minecraft servers. Support for realms is being looked into, but is not guaranteed to be added.

Why does the bot need Manage Channels permission?

The next major update currently in development will allow you to use the names of voice channels to display information live in the sidebar of your server. The invite requests this permission in anticipation of this update. Denying the bot this permission will not affect its functionality at this time.

What does the bot store? "Privacy Policy"

The bot stores the commands issued to it for debugging purposes, and this data is not retained unless neccessary (in the case that it causes a crash.) If you choose to use the saved server option, the bot will store the ID of your Discord server and the IP and type of your Minecraft server. No data stored by the bot will be sold or transferred to any other party.
