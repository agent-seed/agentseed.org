Minecraft Java moderation

I'm sure everyone here had heard about Microsoft's attempts to moderate Minecraft: Java Edition, I'm also sure most of you know that most of the playerbase is very resistive of this change. The new changes in release 1.19.1 will allow player reporting on any server, and banning from any server. To protect this community and myself I am going to use a plugin to prevent reporting when this feature does release. This plugin works by not verifying the identity of the player any message was sent by, therefor making it possible to impersonate someone, because of this Microsoft will not let you report the message. If do not like this, and would rather have a reporting enabled, I suggest you go play on a different server. This is not because I am striving for a toxic server environment, but because I fear this give to trolls and trouble makers. I urge you all to stand out against Microsoft for allowing this.
